Downloads

Currently, we simply separate our DeepAccident dataset to the perception data and motion data for downloading. Each of the downloaded file contains full sensor and annotation information.

We will provide separate download links for different sensor modality and annotations in the future to allow users only downloading the part that they need.

Data organization

Perception data

We have four data collection vehicles for every time step. Each of those vehicles contain a folder for sensor data and annotations, including ego_vehicle, ego_vehicle_behind, other_vehicle, other_vehicle_behind.
